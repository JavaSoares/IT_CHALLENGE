- CSS

---
## 📁 Como baixar o projeto

```bash

# Clonar o repositorio
$ git clone https://wwww.github.com/JavaSoares/IT_CHALLENGE

# Entrar no diretorio 
$ cd IT_CHALLENGE

# Instalar as dependências
$ yarn install

# Iniciar o projeto
$ yarn start
```
